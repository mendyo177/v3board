## 新增功能（简述）
- 自定义节点批量导入：后台支持批量导入原始节点链接，便于集中管理。
- 通用订阅：支持导出多种客户端格式的订阅。
- 本地验证码：本地生成图片/SVG 验证码，减少对外部服务依赖（面板可配置开启/关闭）。
- 后台界面增强：加入导入入口并优化在线状态显示。

## 升级提示
如需回退到原来的外部验证码服务，只需在面板中关闭验证码开关即可。
